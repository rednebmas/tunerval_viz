By default, each colored set of cells represents one user. The coloring can be changed to represent platform (iOS or Android), or the musical interval for which the questions was answered. The splits within the a color represent the number of questions they have answered different intervals. If you wish to see how their scores have progressed over time, click the cell to see their score progression.

Please see the Explatory Analysis which led to the development of this visualization for a detailed description of the data set.
